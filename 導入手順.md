# TimeREX → Google Tasks 自動化　導入手順

費用：完全無料 / 所要時間：約10分

---

## ① GASプロジェクトを開く

1. 使いたいGoogleアカウント（TimeREXメールが届く方）でChromeにログイン
2. ブラウザで以下を開く
   ```
   https://script.google.com
   ```
3. 左上の「新しいプロジェクト」をクリック
4. プロジェクト名を「TimeREX自動化」などに変更（任意）

---

## ② コードを貼り付ける

1. エディタに最初から書いてある `function myFunction() {}` を全部消す
2. ダウンロードした `timerex_to_tasks_free.gs` の中身を全部コピーして貼り付ける
3. 上部の「💾 保存」ボタン（またはCtrl+S）を押す

---

## ③ Tasks APIを有効化する

1. 左メニューの「＋ サービス」をクリック
2. 一覧から「**Google Tasks API**」を探して選択
3. 「追加」をクリック
   → 左メニューに「Tasks」が追加されればOK

---

## ④ セットアップ確認（動作テスト）

### 4-1. 権限を許可する

1. 上部の関数選択ドロップダウンで **「setup」** を選ぶ
2. 「▶ 実行」をクリック
3. 「権限を確認」→ 自分のGoogleアカウントを選択
4. 「詳細」→「TimeREX自動化（安全ではないページ）に移動」→「許可」

   ※ 自分で作ったスクリプトなので安全です

5. 下部の「実行ログ」に以下が表示されればOK
   ```
   ✅ Tasks API: OK
   ✅ Gmail: OK（TimeREXメール ○件検出）
   ```

### 4-2. 解析テスト

1. 関数を **「testParse」** に切り替えて「▶ 実行」
2. ログに以下が出ればOK
   ```
   ✅ 解析成功
   相手: 田中太郎さん
   日時: 29日(月)22:30～
   定型文: お世話になっております。...
   ```

---

## ⑤ トリガーを設定する（自動実行）

1. 左メニューの「⏰ トリガー」をクリック
2. 右下の「**＋ トリガーを追加**」をクリック
3. 以下のように設定する

   | 項目 | 設定値 |
   |------|--------|
   | 実行する関数 | `runTimerexToTasks` |
   | イベントのソース | 時間主導型 |
   | 時間ベースのトリガーのタイプ | 時間ベースのタイマー |
   | 時間の間隔 | **1時間おき**（または30分おき） |

4. 「保存」をクリック

---

## ⑥ 完了！動作の流れ

```
TimeREXからメール着信
        ↓
GASが1時間ごとに自動チェック
        ↓
未処理メールを検出
        ↓
相手・日時・URLを自動解析
        ↓
定型文を生成
        ↓
Google Tasksに「TimeREX予定」リストで登録
        ↓
Gmailに「TimeREX処理済」ラベルを付与（二重処理防止）
```

**Google Tasksのタスク例：**
- タイトル：`【TimeREX】田中太郎さん 29日(月)22:30～`
- メモ：
  ```
  お世話になっております。
  29日(月)22:30～よろしくお願いいたします。
  お時間になりましたらこちらからご入室ください。
  https://meet.google.com/abc-defg-hij
  ```
- 期限：2026年6月29日

## トラブルシューティング

**「Tasks API: ❌」と出た場合**
→ 手順③をもう一度確認。サービス追加後に再実行。

**「TimeREXメール 0件検出」と出た場合**
→ Gmailにまだメールが届いていない可能性あり。届いた後に再テスト。

**日時が解析できない場合**
→ メール本文をコピーしてClaudeに「このメールのフォーマットで正規表現を修正して」と貼り付けて相談。
