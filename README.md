# TimeREX → Google Tasks 自動化ツール

TimeREXの予約通知メールを自動検知し、
返信用の定型文をGoogle Tasksへ自動登録するGASツールです。

## 機能
- GmailでTimeREX通知メールを自動検知
- 相手の名前・日時・会議URLを自動解析
- 返信用定型文を自動生成
- Google Tasksにタスクとして自動登録
- 処理済みメールにラベルを自動付与（二重処理防止）

## 使用技術
- Google Apps Script
- Gmail API
- Google Tasks API
- 正規表現

## 費用
完全無料（外部APIなし）
